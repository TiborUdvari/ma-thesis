you only need to install java to run processing with the "expert" release. if you don't have admin privileges to install java, then you should use the "standard" release.  
 
if you got through the 'javaw' problem with the expert release,  you'd then get the quicktime problem.  
 
so unfortunately we just need to remove the quicktime requirement. as mentioned in the release notes, this is more difficult than expected, but i hope to fix it soon. casey had problems with this recently too so maybe i can bump it up on the priority list. lemme see what i can do.
