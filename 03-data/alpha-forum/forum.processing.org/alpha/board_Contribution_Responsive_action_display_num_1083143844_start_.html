top down shooter

I wrote this system tonight to play around with the idea of side-scrolling shooting games. At first I wanted an r-type style sideways shooter, but I decided my system was better for a top-down type game.
 
http://www.design.ucla.edu/~mflux/p5/firing/applet
 
use your mouse to move the crosshair and fire
 
use first person shooter keys:
"w" "a" "s" "d" to move the ship up down left and right.
 
I used a slight steering mechansim to make it "feel" like the ship is flying.
 
Gonna keep making this into a game.
