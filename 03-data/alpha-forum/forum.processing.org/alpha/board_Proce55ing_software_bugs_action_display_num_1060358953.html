yeah, any number of copies of java might be installed on your machine.  
 
this can be part of the problem though--that if you deinstall one copy, or reinstall another, then the QTJava.zip that was placed inside its ext folder will become irrelevant.. and may not get updated properly until quicktime is reinstalled (since QTJAVA will no longer point to a valid qtjava.zip if it no longer exists)
