I was using many Samples running at the same time. But if you want to use the synthesis with LiveOutput, I guess that you have to do something like StreamA[]+StreamB[] before sending it to LiveOutput.data[].
 
Code:

for(int i = 0; i < LiveOutput.data.length; i++)  
LiveOutput.data[i] = StreamA[i]+StreamB[i];
 
or
Both streams but 50% volume:
 
for(int i = 0; i < LiveOutput.data.length; i++)  
LiveOutput.data[i] = StreamA[i]/2+StreamB[i]/2;

 
I could be wrong, I didn't try, tell me if it sound ok.
