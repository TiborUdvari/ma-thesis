processing is "simplified" java, there a minor parts (0.2 becomes 0.2f for float, color becomes int, ..) that are changed by a preprocessor inside the environment to make it "pure" java, then it's compiled using jikes.
now, to include a sketch class into another java project you'd have to include all of processing (-classes) - if the sketch uses any of the processing buildin objects of functions. you will have to include "pde.jar" and "comm.jar" from processing's lib folder into your project (on windows), then use the ".java" generated by "export to web" in your project.
 
here's a comparison between java and processing:
http://processing.org/reference/compare/java.html
